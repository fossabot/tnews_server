## TNews API
API for tnews.

### Quick start

- Clone projects:

```bash
git clone https://github.com/tnews/tnews_server.git
```

- Get packages from **pub** for Flutter app:

```bash
dart pub get
```

- Run project with **debug** mode:

```bash
dart --observe bin/dev.dart
```

- Run project with **release** mode

```bash
dart bin/prod.dart
```
